# C++ Login System

---

## Beschreibung

Dieses Projekt ist ein einfaches Login-System, das ich als Schulprojekt programmiert habe.  
Der Benutzer gibt Zugangsdaten ein, die anschließend überprüft werden.

---

## Funktionen

- Eingabe von Benutzername und Passwort
- Vergleich mit gespeicherten Daten
- Ausgabe von Erfolg oder Fehler

---

## Was ich gelernt habe

- Arbeiten mit Strings
- Vergleich von Eingaben
- Nutzung von logischen Operatoren
- einfache Programmstruktur
